2shine a skin to make Dnn websites beautiful

2shine is designed to be a skin (so basically a graphical overhaul) for Dnn. If you are not familiar with Dnn you should be starting there. You will get the most out of 2shine in combination with 2sxc which is an UI improvement for Dnn.

2shine is an open-source software which was designed and developed by professionals at 2sic in Switzerland. 2shine is build on Bootstrap 5 and implements all the newest technologies and best practices.
